WHAT IS A TOOL?


  • A tool is basically an html application which can be launched by Minecraft-DT

  • Most of the tools should be baised on the game called minecraft

  • They provide useful mecanics to help out minecraft engineers
    in some basic or complex tasks


Example

One command block generator tool can help the user convert their command
creation in to one summoning command


HOW TO INCLUDE MY TOOL?


  • You can include your tools by pressing "My Tools" button at the bottom right corner
    (A new window should appear on your screen)

  • What you need to do next is to simply drag and drop your tool into this window

  • The last thing you need to do is to restart the application

HOW TO CREATE MY OWN TOOL?


  • To create your own tool, you must learn the following programming launguages;
    HTML, HTA and JavaScript (or jQuery for complex work)

  • Now you need a template for your tool which is located in your tools inventory

  • To access the inventory, go to main screen of Minecraft-DT and press"My Tools"
    button. You should be able to see the folder called "Default"

  • Everything is organised for you, the only thing that you must take focus on is on the
    HTA file called "run" which is your tool display that will appear when launched by
    Minecraft-DT

  • Now you must design your tool. Make a plan of what you want your tool to do
    After the planning, you will be required to do your coding part

  • You can change your tools image which should be located in your tool
    "images" folder

  • To rename your tool, simply rename the folder name in which
    your "run.hta" file is locaded at


Notes

Notice that if you change the location of your "run.hta" file to different folder in the
"Default" folder, your tool would not be displayed in the Minecraft-DT application
